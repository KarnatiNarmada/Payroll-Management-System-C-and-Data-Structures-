Payroll Management System in C

This is a console-based payroll management system developed in the C programming language using data structures. The program allows you to store employee details, calculate salaries, and generate payslips directly from the terminal.

Features:
Add and store employee details
Calculate allowances, deductions, and net salary
Generate a formatted payslip in the console
Search and update employee records
Save employee data in text files for future use

Learning Outcomes:
Use of C structures to manage complex data
Implementation of array and linked list based employee management
Modular programming using separate C and header files
File handling for persistent storage of employee information

Usage:
The program runs in the console. After starting the program, you can log in as an administrator to manage employee records or as an employee to view your payslip and personal information.

Suitable For:
This project is ideal for students learning C programming, data structures, and file handling in systems programming.
